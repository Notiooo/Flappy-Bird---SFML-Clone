Detailed Description

template<typename ResourceType, typename Identifier>
class ResourceManager< ResourceType, Identifier >

This class allows us to hold some textures in it, and get them back later.

I use this function due to the fact that Textures, or sounds are very heavy, and I want to store them once, and them use them multiple times.

Template Parameters
ResourceTypeType of the resource we want to store
IdentifierThe identifiers we want to use when selecting back to our resources.
