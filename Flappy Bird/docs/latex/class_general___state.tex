\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename State\+Type$>$\newline
class General\+\_\+\+State$<$ State\+Type $>$}

An abstract \mbox{\hyperlink{class_general___state}{General\+\_\+\+State}} other states can inherit from. 

This abstract class defines all general behaviours of the State like for example\+: 1) request Push -- push given state onto the stack 2) request Pop -- pops state from the top of the stack 3) request Clear -- clears the stack

Also it defines some typical S\+F\+ML functions like\+: update, draw or handle events.
