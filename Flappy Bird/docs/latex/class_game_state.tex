\doxysubsection{Detailed Description}
A state where the actual game starts. 

The target game that the player can interact with and have fun.

The whole mechanics of the game takes place here. The game world and its objects are kept in this state. There is also another \mbox{\hyperlink{class_state_stack}{State\+Stack}} here which controls flow of states of the bird.
