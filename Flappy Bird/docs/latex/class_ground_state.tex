\doxysubsection{Detailed Description}
This is the condition in which the bird hit the ground. 

In this state the player cannot control the bird. This state displays \char`\"{}\+Game Over\char`\"{}, and waits a few seconds after which it changes State to \char`\"{}\+Score\+State\char`\"{}.
